Términos y descripciones

W3C
Asegura el crecimiento de la web a lo largo del tiempo mediante estándares
WHATWG
Comunidad de personas interesadas en la evolución de HTML. La fundaron personas provinientes de:
  • Fundación Mozilla
  • Opera Software
  • Apple
HTML
Lenguaje de marcado para la creación de páginas web.
XHTML
Es un lenguaje de marcado que se basa en usar un HTML que sea válido en XML.
DTD
Es un documento que describe la estructura y sintaxis de un documento XML o SGML
APIs
Es una pieza de código que permite la intercomunicación entre aplicaciones. Es sobre todo una interfaz Aquí una lista de estas.
  1. Drag and drop: se basa en agarrar y arrastrar un objeto virtual a la ubicación de otro para dar una interacción.
  2. Geolocalización: permite al usuario compartir su ubicación con otras aplicaciones si así lo desea.
  3. Almacenamiento local: permite al usuario guardar datos dígitales en dispositivos de almacenamineto físicos.
  4. App Cache: permite que el contenido y las páginas web sigan disponibles aunque esté en modo offline la apliación web
  5. WebWorkers: permiten que el contenido web ejecute scripts en segundo plano en forma de hilos.
Editor WYSIWYG
Es un Software que permite a usuarios profanos diseñar páginas web HTML donde tu diseñas las cosas gráficamente y el software lo traduce a código HTML. Es ek acrónimo de What You See Is What You Get
Navegadores
Es un programa que se encarga de interpretar los contenidos de la página web. Aquí los navegadores más usados según su cantidad de usuarios:
  1. Safari: Desarrollado por Apple
  2. Opera: propiedad de Golden Brick Capital
  3. Mozilla Firefox: navegador web de código liPbre
  4. Google Chrome: desarrollado por Google apartir de proyectos de código abierto.
