Listado

A
Selecciona todos los elementos de tipo A.

#id
Selecciona todos los elementos con el identificador.

A B
Selecciona los descendientes de tipo B de un elemento A.

.classname
Selecciona los elementos de la clase elegida.

A, B
Combinador de selectores, selecciona los elementos A y elementos B.

*
Selector universal, selecciona todos los elementos.

A + B
Selecciona los hermanos de una clase B directamente seguidos del elemento A.

A ~ B
Seleeciona todos los hermanos de una B seguidos después de una clase A.

A > B
Selecciona todos los elementos de clase B que son hijos directos de una clase A.

:first-child
Selecciona todos el primer hijo de un elemento.

:only-child
Selecciona los elementos que son el único hijo de otros elementos.

:last-child
Selecciona el último hijo de un elemnto.

:nth-child(A)
Selecciona el Aº elemento hijo de otro elemnto.

:nth-last-child(A)
Lo mismo que el nth-child(A) pero contando desde el final.

:first-of-type
Selecciona el primer elemento del tipo especificado en cualquier otro elemento.

:nth-of-type(A)
Selecciona cada Aº elemento de un tipo, selecionando incluso elementos pares e impares.

:nth-of-type(An+B)
Selecciona todos los Aº elementos de un tipo apartir del elemento número B.

:only-of-type
Selecciona el único elemento del tipo seleccionado dentro de otro elemento.

:last-of-type
Selecciona el último elemento del tipo seleccionado dentro de otro elemento.

:empty
Selecciona elementos que no tienen ningún descendiente.

:not(X)
Selecciona todos los elementos que no cumplen el selector X.

[attribute]
Selecciona todos los elementos que posean ese atributo.

A[attribute]
Selecciona todos los elementos A que tienen ese atributo.

[attribute="value"]
Selecciona todos los elementos que tienen ese atributo con ese valor.

[attribute^="value"]
Selecciona todos los elementos que tienen ese atributo y su valor empieza por value.

[attribute$="value"]
Selecciona todos los elementos que tienen ese atributo y su valor termina por value.

[attribute*="value"]
Selecciona todos los elementos que tienen ese atributo y su valor contiene a value.
